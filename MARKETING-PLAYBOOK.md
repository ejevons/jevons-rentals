**AppFolio → Settings → Listing Syndication**

| Platform | Priority | Notes |

| Zillow / Trulia / HotPads | CRITICAL | #1 renter traffic source |
| Apartments.com / Rent.com | CRITICAL | High-intent renters |

| Rentals.com | MEDIUM | Additional exposure |
| Dwellsy | MEDIUM | Growing platform |
| RentPath (Apartment Guide) | MEDIUM | Legacy but still active |

### Listing Quality Checklist (per vacancy):
- [ ] 10+ high-quality photos (exterior, every room, kitchen, bath, amenities)
- [ ] Accurate rent amount and deposit
- [ ] Correct bedroom/bathroom count and square footage
- [ ] Pet policy clearly stated
- [ ] Available date set correctly
- [ ] Full description with amenities, neighborhood highlights, and application link
- [ ] Screening criteria linked or included
- [ ] Virtual tour link (if available)

### Syndication Monitoring:
